Welcome, weary traveler. Here you can rest your wings and imbibe a mug of hot klah. This site is dedicated to dragons and dragon-friends. I guess you can consider it the official web site of the newsgroup Alt.Fan.Dragons, seeing as this site is maintained by somebody who regularly posts there. Maybe not as official as Dragonfire was, since Dragonfire was made by the actual founder of AFD, but that site doesn't appear to exist anymore.

As webmaster, I make these promises to you:

Accessibility
  • This site is proud to use 100% standards-compliant HTML 5.1 and CSS 3. That means any modern web browser should be able to display this site properly.
  • The server can be accessed using IPv4 or v6, so it's ready for when the inevitable switch to IPv6 happens.
  • I am trying to make the site as accessible as possible - that means it should work with screen readers, Braille displays, automatic language translators, and so on.
  • If you do have any problems, you can contribute fixes on GitHub since DragonSmoke.Cloud is completely FLOSS (to the extent that a web site can be considered software) and Free Culture (to the extent that a web site can be considered a cultural work).
  • Nothing on this site will ever be encumbered by DRM.
Privacy & security
  • There are no cookies used, so I cannot track you that way.
  • There is no JavaScript, Flash, or Silverlight, so you can use this site without running software other than your browser.
  • The site is fully TLS encrypted, with forward secrecy and HTTP strict transport security. That way no MITM can eavesdrop on, or modify, the content of these pages when they are transmitted to you.
  • No information is collected beyond what is necessary to run a simple web server, so no information about you can ever leak from this site. No names, no passwords, no credit card info - nothing.

Just because my site is small and niche does not mean I will settle for subpar security.
