Welcome, weary traveler. Here you can rest your wings and imbibe a mug of hot klah. This site is dedicated to dragons and dragon-friends. I guess you can consider it the official web site of the newsgroup Alt.Fan.Dragons, seeing as this site is maintained by somebody who regularly posts there. Maybe not as official as Dragonfire was, since Dragonfire was made by the actual founder of AFD, but that site doesn't appear to exist anymore.

This site is proud to use 100% standards-compliant HTML 5.1 and CSS 3. That means any modern web browser should be able to display this site properly. The server can be accessed using IPv4 or v6, so it's ready for when the inevitable switch to ipv6 happens. There are no cookies used, so I cannot track you that way. There is no JavaScript or Flash, so you can use this site without running software other than your browser. The site is fully TLS encrypted, with forward secrecy and HTTP strict transport security. Just because my site is small and niche does not mean I will settle for subpar security.
